# 项目简介

## 核心功能
- 基于YOLOv8的井盖状态检测系统，支持识别5类井盖状态：损坏(broke)、圆形(circle)、完好(good)、缺失(lose)、 uncovered( uncovered)
- 提供1118张标注数据集，包含完整训练集和测试集
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时检测界面，显示目标位置、目标总数、置信度等信息
- 支持批量检测功能，可查看所有检测结果
- 可自定义检测参数：置信度阈值、IOU阈值
- 提供模型选择功能，支持切换不同检测模型
- 支持检测结果保存功能

## 适用场景
- 市政基础设施巡检
- 井盖状态监测与管理
- 自动化图像检测应用
- 计算机视觉学习与研究
- 深度学习模型部署实践

## 技术特点
- 采用YOLOv8目标检测算法，具有较高的检测精度
- 基于PyQt5开发的图形界面，操作简便直观
- 支持多种输入格式，适应不同应用场景
- 提供完整的源代码和数据集，便于二次开发
- 包含详细的运行教程和环境配置说明
